## 快速开始

### 步骤1：环境检查
```bash
# 检查ROS2环境
ros2 --version
ros2 topic list
```

### 步骤2：创建项目
```bash
cd ~/ros2_ws/src
ros2 pkg create --build-type ament_cmake smart_car_core_demo \
  --dependencies rclcpp geometry_msgs std_msgs std_srvs rclcpp_action
```

### 步骤3：复制代码
将本目录中的代码复制到您的项目中，具体步骤参见 `05_快速开始_编译运行指南.md`

### 步骤4：编译运行
```bash
cd ~/ros2_ws
colcon build --packages-select smart_car_core_demo
source install/setup.bash

# 测试话题功能
ros2 run smart_car_core_demo speed_publisher

# 测试服务功能
ros2 run smart_car_core_demo emergency_stop_server

# 测试动作功能
ros2 run smart_car_core_demo countdown_park_server
```

## 常见问题

### Q1：编译时找不到头文件？
A：确保已安装所有依赖，参见 `05_快速开始_编译运行指南.md` 第3节

### Q2：节点启动后没有输出？
A：检查环境变量和包编译，使用 `ros2 node list` 确认节点是否运行

### Q3：话题无法通信？
A：检查话题名称和类型，使用 `ros2 topic list` 和 `ros2 topic echo` 调试

### Q4：如何在真实智能车上运行？
A：参见 `05_快速开始_编译运行指南.md` 第5节，需要根据实际硬件调整参数
