## 故障排除

### 常见问题
1. **找不到 ament_cmake**
   - 原因: 未正确 source ROS2 环境
   - 解决: `source /opt/ros/humble/setup.bash`

2. **找不到 ros_robot_controller_msgs**
   - 原因: 依赖包未构建或未 source 工作空间环境
   - 解决: 先构建消息包，然后 `source install/local_setup.bash`

3. **Python 包未显示在 ros2 pkg list 中**
   - 原因: Python 包的注册方式不同
   - 解决: 包已正确安装，可通过 Python import 验证
