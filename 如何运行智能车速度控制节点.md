# 智能车速度控制节点运行指南

本文档详细说明如何成功运行 `speed_publisher.cpp` 和 `speed_subscriber.cpp` 两个ROS2节点。

## 项目概述

本项目包含两个ROS2节点，用于智能车速度控制：
1. **speed_publisher** - 发布速度指令到 `/cmd_vel` 话题
2. **speed_subscriber** - 订阅 `/cmd_vel` 话题，计算电机转速并发布到 `/ros_robot_controller/set_motor` 话题

## 系统要求

- Ubuntu 22.04 LTS
- ROS2 Humble
- 已安装 colcon 构建工具

## 项目结构

```
/home/ubuntu/ros2_wsorincp/
├── src/
│   └── ros_robot_controller_msgs/    # 自定义消息包
└── ros2_core_examples/
    └── smart_car_core_demo/          # 智能车演示包
        ├── src/
        │   └── smart_car_core_demo/
        │       ├── speed_publisher.cpp
        │       └── speed_subscriber.cpp
        ├── CMakeLists.txt
        └── package.xml
```

## 构建步骤

### 1. 准备工作空间环境

```bash
cd /home/ubuntu/ros2_wsorincp
source /opt/ros/humble/setup.bash
```

### 2. 构建依赖包

首先构建自定义消息包：

```bash
colcon build --packages-select ros_robot_controller_msgs
```

### 3. 构建智能车演示包

```bash
source install/setup.bash
colcon build --packages-select smart_car_core_demo
```

### 4. 激活工作空间

```bash
source install/setup.bash
```

## 运行节点

### 方法一：分别运行两个节点（推荐）

#### 终端1：运行速度订阅节点

```bash
cd /home/ubuntu/ros2_wsorincp
source install/setup.bash
ros2 run smart_car_core_demo speed_subscriber
```

预期输出：
```
[INFO] [时间戳] [speed_subscriber]: 速度订阅节点已启动，等待cmd_vel指令...
[INFO] [时间戳] [speed_subscriber]: 速度订阅节点运行中...
```

#### 终端2：运行速度发布节点

```bash
cd /home/ubuntu/ros2_wsorincp
source install/setup.bash
ros2 run smart_car_core_demo speed_publisher
```

预期输出（每100ms一次）：
```
[INFO] [时间戳] [speed_publisher]: 发布车速度指令：线速度=0.50 m/s, 角速度=0.20 rad/s
```

### 方法二：使用launch文件（如果可用）

如果存在launch文件，可以使用：

```bash
ros2 launch smart_car_core_demo topic_demo.launch.py
```

## 节点功能说明

### speed_publisher 节点

- **功能**：定时发布固定的速度指令
- **发布话题**：`/cmd_vel` (geometry_msgs/Twist)
- **发布频率**：10Hz (100ms间隔)
- **速度值**：
  - 线速度：0.5 m/s (X方向)
  - 角速度：0.2 rad/s (Z方向)

### speed_subscriber 节点

- **功能**：
  1. 订阅 `/cmd_vel` 话题
  2. 使用麦克纳姆轮运动学计算四个电机的转速
  3. 发布电机转速到 `/ros_robot_controller/set_motor` 话题
- **订阅话题**：`/cmd_vel`
- **发布话题**：`/ros_robot_controller/set_motor` (ros_robot_controller_msgs/MotorsState)
- **运动学参数**：
  - 前后轴距：0.216m
  - 左右轴距：0.195m
  - 轮子直径：0.097m

## 故障排除

### 常见问题1：找不到包

**错误信息**：
```
Package 'smart_car_core_demo' not found
```

**解决方案**：
```bash
cd /home/ubuntu/ros2_wsorincp
source install/setup.bash
```

### 常见问题2：缺少消息依赖

**错误信息**：
```
fatal error: ros_robot_controller_msgs/msg/motor_state.hpp: No such file or directory
```

**解决方案**：
1. 确保 `ros_robot_controller_msgs` 包在正确位置：`/home/ubuntu/ros2_wsorincp/src/`
2. 重新构建两个包：
   ```bash
   colcon build --packages-select ros_robot_controller_msgs smart_car_core_demo
   ```

### 常见问题3：CMake找不到包

**错误信息**：
```
Could not find a package configuration file provided by "ros_robot_controller_msgs"
```

**解决方案**：
在构建 `smart_car_core_demo` 之前，确保已source安装文件：
```bash
source install/setup.bash
```

## 停止节点

### 方法一：使用Ctrl+C
在每个运行节点的终端中按 `Ctrl+C`

### 方法二：使用pkill命令
```bash
pkill -f "speed_publisher\|speed_subscriber"
```

### 方法三：停止所有ROS2节点
```bash
pkill -f ros2
```

## 重新启动

如果需要重新启动节点，只需重复"运行节点"部分的步骤。

## 扩展功能

### 修改速度值
要修改发布的速度值，编辑 `speed_publisher.cpp` 文件中的以下行：
```cpp
message.linear.x = 0.5;  // 修改线速度
message.angular.z = 0.2; // 修改角速度
```

### 修改发布频率
修改 `speed_publisher.cpp` 中的定时器间隔：
```cpp
timer_ = this->create_wall_timer(
    100ms,  // 修改此值，如：500ms 为 2Hz
    std::bind(&SpeedPublisher::timer_callback, this)
);
```
